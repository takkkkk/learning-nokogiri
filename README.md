# learning-nokogiri: Nokogiriの練習
これはcommitの練習です。
## 始め方

事前にVagrantにログインしておく。

```
$ vagrant up
$ vagrant ssh
```

ログイン先の作業:

```
[vagrant@jessie: ~] $ git clone https://github.com/kd-tech-rails2016/learning-nokogiri.git
[vagrant@jessie: ~] $ cd learning-nokogiri
```

## 実行

```
[vagrant@jessie: ~] $ bundle exec ruby debiancode.rb
```
